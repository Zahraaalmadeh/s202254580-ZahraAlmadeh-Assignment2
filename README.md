# s202254580-ZahraAlmadeh-Assignment2
## 📌 Project Description
This project is a personal portfolio website developed as part of Assignment 2. It builds on Assignment 1 by adding interactivity, dynamic content, and improved user experience using JavaScript and modern web design techniques.

The website showcases personal information, projects, and a contact form.

---

## ✨ Features
- Responsive and modern portfolio layout
- Dark / Light mode toggle (saved using localStorage)
- Project filtering by category (interactive feature)
- Contact form with validation and feedback message
- Smooth hover effects and transitions
- Dynamic footer year update
- User-friendly design and layout
