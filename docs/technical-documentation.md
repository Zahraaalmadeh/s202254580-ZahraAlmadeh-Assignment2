# Technical Documentation

## 📌 Overview
This portfolio website was developed using HTML, CSS, and JavaScript. It includes an About section, Projects section, and Contact section, along with interactive features and improved user experience.

---

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## ⚙️ Dynamic Features

### 1. Theme Toggle
- Users can switch between light and dark mode
- The selected theme is saved using localStorage
- Theme persists after page reload

### 2. Project Filtering
- Users can filter projects by category (Web, Finance, All)
- Projects update dynamically based on user selection
- If no projects match, a message is displayed

### 3. Contact Form Validation
- Checks if all fields are filled
- Displays error message if inputs are empty
- Displays success message when submitted correctly

---

## ❗ Error Handling

The application provides clear feedback to users:

- "Please fill in all fields" → for empty form inputs
- "Message sent successfully" → for valid submission
- "No projects found" → when filter returns no results

---

## 🎨 User Experience Enhancements

- Smooth hover effects on buttons and project cards
- Responsive design for mobile and desktop
- Clean layout with centered content
- Gradient backgrounds for modern appearance
- Interactive feedback messages
