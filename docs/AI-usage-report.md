# AI Usage Report

## 🤖 AI Tools Used
- ChatGPT

## 🧠 How AI Was Used
ChatGPT was used throughout the development of this project to:

- Improve the layout and visual design using CSS
- Generate and simplify JavaScript for:
  - Dark/light mode toggle
  - Contact form validation
  - Project filtering feature
- Help debug Git and GitHub issues
- Suggest ideas for interactive features
- Assist in writing documentation (README and reports)

---

## ✏️ Modifications and Improvements
I did not directly copy AI-generated code without review. Instead, I:

- Modified and simplified JavaScript to better understand it
- Adjusted styling to match my personal design preferences
- Integrated AI suggestions into my existing code structure
- Tested all features to ensure they work correctly

---

## 📚 What I Learned
Using AI tools helped me learn:

- How to implement dynamic content using JavaScript
- How to use localStorage to save user preferences
- How to validate forms and display feedback messages
- How to structure a portfolio project properly
- How to improve UI/UX with small interactive features

---

## ⚠️ Reflection
AI tools were helpful in speeding up development and solving problems, but I made sure to understand all code used in the project and adapted it to fit the assignment requirements.
